# Free Expense Splitter

A simple client-side expense splitting tool that calculates optimized
settlements and minimizes transactions.

Built as a single standalone HTML file. No backend required.

## Features

-   Add participants and expenses
-   Automatic balance calculation
-   Simplified settlement generation
-   Fully client-side

## Usage

Download the HTML file and open it in your browser to start splitting
expenses. 

Or just visit this [site](https://devamshi.github.io/free-expense-splitter/)
